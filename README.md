# Echo Hunter 🚀

Ultra-lightweight Android game built with custom Canvas rendering. No frameworks, pure Activity-based architecture, optimized for a tiny APK size.

## 🎮 About The Game

### **The Story: Protocol Probe-7**
System corruption has reached a critical **98%**. You are **Probe-7**—the last line of digital defense. Your mission is to purge the sectors and eliminate the growing virus before the mainframe is lost forever. But beware: the virus is evolving, learning from your moves, and the deeper you go, the more you risk becoming corrupted yourself. Will you emerge as the new core protocol, or will you be deleted?

### **The Objective**
The primary goal is to navigate the high-speed corridors of the mainframe, hunting down **Mutating Sector Guardians** while maintaining system integrity.
* **Collect Data:** Secure lost Data Fragments (Score) to reach sector targets and advance.
* **Pulse & Reveal:** Use your Radar Pulse to reveal hidden threats and build high-scoring combos.
* **Avoid Corruption:** Dodge or destroy Red Corrupted AI Entities and escape the deletion firewall.
* **Overclock:** Channel accumulated energy into 'Overclock' mode to turn the tide against the corruption.

## 🛠️ Under The Hood (Tech Stack)
This project is an experiment in extreme optimization and zero-allocation game loops on Android:
* **No Game Engines:** Built entirely using native Android APIs.
* **Custom Rendering:** Everything is drawn directly onto a Canvas.
* **Zero External Assets:** No images, no MP3/WAV files. UI is pure math and primitive shapes.
* **Object Pooling:** Uses primitive arrays for entities to prevent GC pauses.
